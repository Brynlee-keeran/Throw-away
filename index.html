Application Overview:

Students and parents are able to view the session available for sign-ups, what classes, times, and price amount. Able to book with button that takes you to Facebook messenger (our main booking way as of now).

Technology Overview:

I used Javacript for the engaging features and CSS to make it all look appealing.

Feature List:

Colorful and appealing CSS, help button, description button, drop-down buttons for info., advanced "last updated" section, monthly and weekly looks into schedule.

Future Featere List

Basic Technical Features:

CSS using the style tags to make the website look good to the eye.

Using the toggle JavaScript to show and hide the price of each class.

Using the div tag to put words and stuff exactly where I want them.

The if/else statements check which section the user wants to view and display the appropriate content.

Using the href to take the user to the place quicker for booking.
